My Projects

As part of my education the main task of each semester is to develop a software system within a given group. So far this has mounted in three different projects with the first being developed in C, the second in JavaScript, the third in C# and the fourth in Java. An important thing to note is that all projects must solve a real life issue to develop our skills in solving actual problems and working in teams as you would at real company.

C - 1st Semester

The purpose of the first semester was to learn imperative programming by developing a smaller system in C. It was decided that the system should help upcoming students in finding the right education. The need for this program stems from the fact that over time, many educations have seen the day of life hence the process of finding the right one becomes harder and harder.

JavaScript - 2nd Semester

In the second semester we were to build a web-application. Again it had to stem from a real life problem which lead to the development of a group movie recommender system meaning that the system would take multiple peoples' preferences into account instead of just one person. This project was developed before Netflix Party, but could be a great addition to it.

C# - 3rd Semester

The third semester set out to teach me object oriented programming with basis in C#. This project presented a new and exciting challenge as we had to build an application for a company for them to use. This resulted in us developing an app to a company called Sportstiming that would facilitate their premium membership; Club Sportstiming.

Java - 4th Semester

For the fourth semester was the most challenging, but also most giving semester as we had to make a compiler for a new programming language defined by ourselves. many different ideas were discussed, but due to our interest, but limited knowledge in functional programming we set out to make a functional language called Buff. Buff is a mathematical purpose language which set to build bridge for a beginner programmer by resembling mathematical notation. The compiler was written in Java using the ANTLR framework.
